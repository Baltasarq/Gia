Ayuda Generador Integral de Aplicaciones Java

(GIA Java)

Parte de IDE de Java


Desde la parte de IDE de Java podemos realizar modificaciones en el código generado a partir del proyecto creado en la parte de diseño.

En la interfaz de la parte de IDE de Java podemos encontrar las siguientes operaciones:

  • Cambiar de vista:

    Mediante esta opción podremos volver a la vista de diseño para realizar las operaciones que deseemos en nuestro proyecto.


  • Añadir archivo java al proyecto

    Se crea un nuevo de código java y se añade a nuestro proyecto.


  • Borrar el fichero seleccionado

    Seleccionamos un documento java del árbol de clases y lo eliminamos del proyecto.Automáticamente se borrará de disco.


  • Guardar archivo:

    Se almacena en disco el archivo de la pestaña activa en ese momento.


  • Guardar Todos los archivos abiertos:

    Se realiza un salvado de todos los archivos abiertos en las pestañas del proyecto.


  • Imprimir código de la pestaña actual:

    El codigo de la pestaña actual se envía a la impresora por defecto del sistema, permitiendo elegir preferencias de impresión comunes.


  • Abrir carpeta contenedora de archivos *.java:

    Abre la carpeta en la que se encuentran todos los archivos generados por la aplicación y creados para el proyecto.


  • Guardar y compilar código de la pestaña actual:

    Se guardan las modificaciones realizadas en el archivo de la pestaña actual y posteriormente se compila el unicamente el archivo actual.


  • Compilar todo el proyecto generado:

    Se compilan todos los archivos *.java del proyecto.


  • Ejecutar el proyecto generado:

    Se lanza la aplicación generada.


  • Ayuda:

    Opción desde la que se puede acceder a una pequeña explicación acerca de cada una de las opciones de la interfaz tanto de la parte de diseño como de la parte de IDE de Java.


  • Generar documentación Java:

    Mediante esta opción se permite que el usuario pueda generar automáticamente la documentación javadoc del proyecto creado.


  • Menú ocultar menu principal
  • Menú Edición
    • Cortar:

      Permite cortar texto del panel de código. También se corresponde con el comando ctrl+x

    • Copiar:

      Permite copiar texto del panel de código. También se corresponde con el comando ctrl+c

    • Pegar:

      Permite cortar texto del panel de código. También se corresponde con el comando ctrl+v

    • Editar Fuente:

      Permite modificar el tamaño y tipo de la fuente seleccionada.

  • Menu Acerca de:

    Menú que muestra información acerca de la aplicación: descripción, autor, año y versión.
